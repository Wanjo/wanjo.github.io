List Usage
Unordered List An unordered list is used generally to present a list of items that do not necessarily need to be shown in any specific order. To start this list you use the tag <ul>.
Ordered List An ordered list is used generally to present a list of items that need to be shown in a specific order - Example: steps to be followed to create a web page called hw4.html. To start this list you use the tag <ol>.
Definition List A definition list is used for glossaries, vocabulary. questions & answers. etc. To start this list you use the tag <dl>.

Definition section

Definiton list

HTML
Hypertext Markup Language - A nonproprietary language interpreted by the web browser. It came from SGML which was more complex, but was well known by Time Berners-Lee that then created the HTML with the intention of sharing research papers over the internet.
CSS
Cascading Style Sheet - A non-programming language used by web developers to style/format their web pages.
FTP
File Transfer Protocol - The Internet protocol that transfers files between computers connected to the Internet.
SSH
Secure Shell - A network communication protocol that enables two computers to communicate and share data.
